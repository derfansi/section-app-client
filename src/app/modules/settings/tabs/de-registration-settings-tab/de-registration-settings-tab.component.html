Registration Settings

Define here the settings for the registration process. You can define how long participants can de register, move their registration and how much money they get back.

Free events

This is the default setting for free events. It can be overridden for each event.

Settings for participants

De Registration is allowed De registration deadline in days before the start of the event Has to be a positive number
Sending the event to another person is allowed Event sending deadline in days before the start of the event Has to be a positive number

Settings for organizers

De Registration is allowed De registration deadline in days before the start of the event Has to be a positive number

Paid events

This is the default setting for paid events. It can be overridden for each event.

Global settings for participants of paid events

De Registration is allowed De registration deadline in days before the start of the event Has to be a positive number Payment fees are refunded on de registration Sending the event to another person is allowed Event sending deadline in days before the start of the event Has to be a positive number Payment fees are refunded on move

Global settings for organizers of paid events

De Registration is allowed De registration deadline in days before the start of the event Has to be a positive number
